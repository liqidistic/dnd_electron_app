# dnd-electron-app

Application **Electron** pour la gestion de tables **Dungeons & Dragons**, avec base **MySQL**.

## Démarrage rapide

1. `npm install`
2. Copier `.env.example` vers `.env` et configurer la connexion MySQL
3. `npm start`
